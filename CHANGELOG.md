# Changelog

## [Unreleased] - UI Improvements

### Added
- New flexible Button component with:
  - Four size options (XSmall, Small, Medium, Large)
  - Four variants (Primary, Secondary, Destructive, Ghost)
  - Icon support (left/right positioning)
  - Loading states with spinner
  - Full width option
  - Improved accessibility with ARIA attributes

- Enhanced Theme switcher:
  - System theme detection and sync
  - Improved keyboard accessibility
  - Better focus management
  - Smooth transitions
  - ARIA labels and states

- Responsive layout improvements:
  - Flexible container widths
  - Better mobile padding
  - Improved spacing across breakpoints
  - Card-style content containers

- Accessibility enhancements:
  - Skip links for keyboard navigation
  - Focus indicators for interactive elements
  - ARIA labels and roles
  - Better keyboard navigation support

- Design system foundations:
  - CSS custom properties for consistent theming
  - Standardized spacing scale
  - Typography system
  - Animation durations
  - Z-index scale
  - Reduced motion support

### Changed
- Updated layout to be more responsive across different screen sizes
- Improved button component with more flexibility and options
- Enhanced theme switching with better system integration
- Standardized focus styles across interactive elements

### Technical Improvements
- Added CSS custom properties for better maintainability
- Implemented consistent spacing system
- Better TypeScript type definitions
- Improved component composition
- Enhanced dark mode implementation
